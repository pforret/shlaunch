# shlaunch

Run applications from CLI

## 🚀 Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/shlaunch

or with `git`

	$ git clone https://github.com/pforret/shlaunch.git
	$ cd shlaunch

## 🔥 Usage

	> shlaunch.sh [action]
